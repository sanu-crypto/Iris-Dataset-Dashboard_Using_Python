Iris Analytics Dashboard

The Iris Analytics Dashboard is an interactive data visualization project developed using Python, Dash, and Plotly. The dashboard analyzes the famous Iris flower dataset and provides meaningful insights through various charts, KPI cards, and statistical visualizations.

The project includes species distribution analysis, feature comparison, correlation analysis, scatter matrix visualization, and outlier detection using the IQR method. It helps users understand relationships between sepal and petal measurements of different Iris flower species in an easy and interactive way.

Features
Interactive Dashboard using Dash
KPI Cards for dataset overview
Species Distribution Pie Chart
Box Plot Visualization
Scatter Matrix Analysis
Correlation Heatmap
Outlier Detection
Responsive Web Interface
